# Canvas Image Slider

This is an example of how to create an HTML 5 Canvas API based image slider

## To Run Locally

1. clone `https://github.com/kimfucious/canvas-image-slider.git` to a directory of your choosing
2. navigate into that directory
3. execute `npm i`
4. execute `npm start`
